3 사이트 전체 구성

이 사이트는 아래 6개의 화면으로 구성되어 있습니다. 화면 사이의 이동은 버튼 클릭 또는 표 안의 글자(제품명, 해외제조업소, 제조국, 제조사명) 클릭으로 이루어집니다. 뒤로 갈 때는 각 화면 위쪽의 "← 돌아가기" 글자를 클릭합니다.

화면 이름이동 방법확인할 수 있는 정보
① 메인 대시보드 사이트에 접속하면 가장 먼저 나오는 화면 전체 수입 / OEM 이력 목록 (제품 단위)
② 공장별 보기 메인 대시보드 위쪽의 "🏭 공장별로 보기" 버튼 클릭 같은 이력을 해외제조업소(공장) 단위로 묶어서 확인
③ 국가별 보기 메인 대시보드 또는 공장별 보기 위쪽의 "🌍 국가별로 보기" 버튼 클릭, 또는 표의 제조국 글자 클릭 국가별 요약, 주요 수입 품목, 해당 국가 제조사 목록
④ 제품 상세 (제조사 후보 화면) 메인 대시보드나 공장별 보기 표에서 제품명 글자 클릭 해당 제품을 취급하는 해외 제조업체 후보 목록
⑤ 제조사 상세 화면 표의 해외제조업소 글자 클릭, 또는 제조사 후보/제조사 목록에서 제조사명 클릭 연락처, 상품 정보, OEM/소싱 정보, 국내수입 추이, 취급 제품 목록
⑥ 수입횟수 추이 창 표의 "📈 추이 보기" 버튼 클릭 연도별 / 월별 수입횟수
화면을 이동한 뒤 다시 처음으로 돌아가려면, 화면 위쪽의 "← 수입/OEM SKU 이력으로 돌아가기"(또는 상황에 맞는 문구)를 클릭합니다. 브라우저의 새로고침(F5) 버튼을 누르면 항상 메인 대시보드 화면으로 돌아갑니다.
4 메인 대시보드

사이트에 접속하면 가장 먼저 보이는 화면입니다. 전체 수입 / OEM 이력이 제품(SKU) 단위 표로 표시됩니다.

표의 한 행은 제품(SKU) 하나가 특정 국내 수입업체에게 납품되는 이력을 의미합니다. 같은 제조사의 같은 제품이라도 납품받는 수입업체가 다르면 서로 다른 행으로 표시됩니다.
메인 대시보드 전체 화면

상단 버튼 영역

상단 배너와 버튼 영역
1🌍 국가별로 보기 — 국가별 보기 화면으로 이동합니다.
2🏭 공장별로 보기 — 공장별 보기 화면으로 이동합니다.
3📤 Excel 업로드 — 수입 이력 엑셀 파일을 올립니다. 자세한 내용은 21페이지 10번 항목을 참고합니다.
4📇 연락처 보강 업로드 — 제조사 연락처(이메일, 홈페이지 등)를 올립니다. 자세한 내용은 21페이지 10번 항목을 참고합니다.
5경쟁사 카드 — 전체 / 코스트코 / 이마트 / 롯데마트 / 홈플러스 / 쿠팡 중 하나를 클릭하면, 아래 표가 해당 유통사가 직접 수입한 제품 이력만 보여주도록 바뀝니다. 다시 전체를 클릭하면 원래대로 돌아갑니다.

검색창 · 날짜 필터 · CSV

검색 및 필터 도구 모음
1검색창 — 제품명, 해외제조업소, MC, 수입업체, 제조국 중 어떤 값으로도 검색할 수 있습니다. 글자를 입력하면 아래 표가 자동으로 검색 결과로 바뀝니다.
2날짜 필터 — 왼쪽 칸에 시작일, 오른쪽 칸에 종료일을 선택합니다. 날짜를 입력하면 표에는 그 기간에 해당하는 이력만 표시됩니다. 입력한 날짜 옆의 ✕ 를 누르면 날짜 조건이 지워지고 표가 전체 기간 기준으로 되돌아갑니다.
3⬇ CSV — 현재 검색 · 필터 조건이 적용된 전체 목록을 CSV 파일로 내려받습니다. 화면에 보이는 행뿐 아니라 조건에 맞는 전체 데이터가 포함됩니다.

검색창 위쪽에는 "총 N건 중 표시" 문구가 표시되어, 현재 조건에 맞는 전체 건수를 확인할 수 있습니다.

표 컬럼 설명

표 영역 (첫 번째 행 기준으로 클릭 가능한 열과 추이 보기 버튼을 표시)
1제품명 — 글자를 클릭하면 해당 제품을 취급하는 제조사 후보 화면으로 이동합니다. (14페이지 7번 항목 참고)
2해외제조업소 — 글자를 클릭하면 그 제조사의 상세 화면으로 이동합니다. (16페이지 8번 항목 참고)
3제조국 — 글자를 클릭하면 그 국가의 상세 화면으로 이동합니다. (11페이지 6번 항목 참고)
4추이 보기 — 클릭하면 그 제품의 연도별 / 월별 수입횟수를 보여주는 창이 열립니다. (20페이지 9번 항목 참고)

열 머리글의 필터 아이콘 (▾)

"제조국" 열 머리글의 ▾ 아이콘 클릭 시 나오는 창
1정렬 — 오름차순 또는 내림차순으로 표를 정렬합니다.
2검색창 — 목록에 나온 값 중 원하는 값을 빠르게 찾습니다.
3값 목록 체크박스 — 원하는 값에 체크하면, 표에는 체크한 값에 해당하는 행만 표시됩니다. 값이 비어 있는 경우 "(비어있음)"으로 표시됩니다.
4확인 / 취소 — "확인"을 누르면 선택한 조건이 표에 적용되고, "취소"를 누르면 창이 닫히고 선택이 취소됩니다.

모든 열 머리글(구분, MC, 제품명, OEM/수입, 수입업체, 해외제조업소, 제조국 등)에서 같은 방식으로 정렬과 필터를 사용할 수 있습니다.

추이 보기 버튼

"추이 보기" 클릭 시 나오는 창
1연도별 수입횟수 표 — 2021년부터 올해까지 연도별 수입 건수를 보여줍니다. 각 값 아래에는 전년 대비 증감률이 함께 표시됩니다.
2월별 수입횟수 그래프 — 월 단위 수입 건수를 선 그래프로 보여줍니다. 그래프 위쪽에는 해당 기간의 총 수입횟수가 표시됩니다.
3기간 필터 — 조회할 시작 월과 종료 월을 선택하면 아래 월별 그래프만 그 기간 기준으로 다시 그려집니다. 위쪽 연도별 표는 이 필터의 영향을 받지 않습니다.
4닫기(✕) — 창을 닫습니다. 창 바깥의 어두운 영역을 클릭해도 닫힙니다.

이 창은 메인 대시보드, 공장별 보기, 제조사 상세 화면의 취급 제품 목록에서 동일한 방식으로 열립니다. 자세한 내용은 20페이지 9번 항목을 참고합니다.

5 공장별 보기

메인 대시보드와 같은 수입 이력을 제품(SKU) 단위로 확인하는 화면입니다. 다만 메인 대시보드처럼 수입업체별로 행을 나누지 않고, 같은 해외제조업소가 취급한 같은 제품이면 수입업체가 여러 곳이어도 하나의 행으로 합쳐서 보여줍니다.

공장별 보기 화면 전체

진입 방법

메인 대시보드 위쪽의 "🏭 공장별로 보기" 버튼을 클릭하면 이 화면으로 이동합니다.

메인 대시보드와 다른 점

  • 카드 제목이 "수입/OEM SKU 이력 (공장별 보기)"로 표시됩니다.
  • 상단 버튼이 "📋 수입이력 별로 보기"(메인 대시보드로 이동), "🌍 국가별로 보기" 로 구성됩니다.
  • 같은 제조사의 같은 제품을 여러 수입업체가 납품받는 경우, 메인 대시보드에서는 수입업체별로 행이 나뉘지만 공장별 보기에서는 한 행으로 합쳐지고 수입업체 열에 5대 유통사(이마트, 홈플러스, 롯데마트, 쿠팡, 코스트코) 배지와 "외 N개" 형태로 함께 표시됩니다.
공장별 보기 상단 버튼 영역
1📋 수입이력 별로 보기 — 메인 대시보드로 돌아갑니다.
2🌍 국가별로 보기 — 국가별 보기 화면으로 이동합니다.
3📤 Excel 업로드 — 메인 대시보드와 동일하게 동작합니다.
4📇 연락처 보강 업로드 — 메인 대시보드와 동일하게 동작합니다.

표에 표시되는 정보

표의 열 구성(구분, MC, 제품명, OEM/수입, 수입업체, 해외제조업소, 제조국, 수입횟수(전체), 연도별 수입횟수, 수입횟수 추이)은 메인 대시보드와 동일합니다. 검색창, 날짜 필터, CSV, 추이 보기 버튼도 메인 대시보드와 같은 방식으로 동작합니다. 자세한 사용법은 4페이지 4번 항목을 참고합니다.

6 국가별 보기

국가 단위로 제조사 현황과 수입 품목을 확인하는 화면입니다.

국가별 보기 지도 화면
1국가명 검색창 — 국가 이름을 입력하면 목록이 나타나고, 원하는 국가를 클릭하거나 첫 번째 항목에서 엔터 키를 누르면 해당 국가 상세 화면으로 이동합니다.

진입 방법

메인 대시보드 또는 공장별 보기 화면 위쪽의 "🌍 국가별로 보기" 버튼을 클릭하면 이 화면으로 이동합니다.

지도에서 국가 선택하는 방법

  • 지도에서 초록색으로 표시된 국가는 데이터베이스에 등록된 국가입니다. 이 국가를 클릭하면 국가 상세 화면으로 이동합니다.
  • 회색으로 표시된 국가는 등록된 데이터가 없어 클릭해도 이동하지 않습니다.
  • 지도 위에 마우스를 올리면 국가명과, 등록된 국가의 경우 순위 · 비중 · 주요 수입품목 정보가 함께 표시됩니다.
  • 지도 오른쪽 위의 +, −, ⟲ 버튼으로 지도를 확대, 축소, 초기화할 수 있습니다.
  • 지도 왼쪽 아래에는 대한민국 수입금액 기준 국가 순위 목록이 항상 표시됩니다.

국가 상세 화면에서 보이는 정보

국가 요약

국가 요약 영역 (예시: 베트남)

국가명과 국기, 대한민국 수입금액 기준 국가 순위와 비중, 등록된 제조사 수, 수입이력 건수가 표시됩니다. 오른쪽에는 상위 국가들의 수입금액 비중 목록이 함께 표시됩니다.

종합점수 안내 — 이 목록의 종합점수는 해당 제조사가 취급하는 여러 SKU 중 가장 높은 점수를 받은 SKU 기준으로 표시됩니다. 대표 제품명으로 함께 표시되는 제품이 바로 그 SKU입니다.
종합등급(A / B / C) 계산 방식 — 종합등급은 아래 3가지 항목을 각각 A(3점) / B(2점) / C(1점)로 평가한 뒤 가중치를 적용해 계산합니다.
  • 주요 유통사 거래 다양성 (가중치 50%) — 이마트 · 홈플러스 · 롯데마트 · 쿠팡 · 코스트코 중 3곳 이상과 거래하면 A, 1~2곳이면 B, 거래가 없으면 C
  • 국내 수입횟수 (가중치 30%) — 동일 제품을 생산하는 제조사들 사이에서 수입횟수가 상위 25% 안에 들면 A, 하위 25%면 C, 그 사이는 B
  • 최근 3개년 성장추세 (가중치 20%) — 최근 3개 완료 연도의 수입횟수가 매년 늘었으면 A, 매년 줄었으면 C, 그 외에는 B
이 3가지 항목을 가중 합산한 점수가 80점 이상이면 A, 50점 이상이면 B, 그 미만이면 C 등급 배지로 표시됩니다.

10 업로드 버튼

데이터를 올리는 버튼들은 메인 대시보드와 공장별 보기 화면 위쪽에 있습니다.

Excel 업로드 버튼

"📤 Excel 업로드" 버튼을 클릭하면 파일 선택 창이 열립니다. 기본 수입 / OEM 데이터가 담긴 .xlsx 파일을 선택하면 업로드가 진행됩니다. 업로드가 끝나면 화면 위쪽에 처리 결과 문구(적재 건수, 스킵 건수)가 표시되고, 표 내용이 새로 고쳐집니다. .xlsx 또는 .xls가 아닌 파일은 업로드되지 않습니다.

연락처 보강 업로드 버튼

"📇 연락처 보강 업로드" 버튼을 클릭하면 파일 선택 창이 열립니다. 제조사의 이메일, 홈페이지, 인증서 등 연락처 정보가 담긴 .xlsx 파일을 선택하면, 기존에 등록된 수입 이력에 해당 연락처 정보가 채워집니다. 업로드가 끝나면 처리 결과 문구(처리한 행 수, 매칭된 행 수)가 표시됩니다.

업로드 후 반영 시간

업로드가 끝나면 화면의 표 내용은 자동으로 새로 고쳐집니다. 다만 업로드한 데이터 양이 많은 경우, 화면 전체(요약 숫자, 국가별 정보 등)에 반영되기까지 몇 분 정도 시간이 걸릴 수 있습니다.
